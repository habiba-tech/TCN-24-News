## 📌 About The Project

**TCN-24 (Time City News)** is a community-based digital news platform.  
This project transforms manual social media publishing into a structured **News Management System** with secure backend integration.

---

## 🎯 Problem Statement

The existing system relied only on YouTube and social media for publishing news.

Issues:
- No centralized database
- No structured categorization
- No authentication control
- Manual archive search
- Security risks

This project provides a full-stack solution to overcome these limitations.

---

## 🚀 Key Features

- 🔐 Secure Admin Login (JWT Authentication)
- 📝 Add / Edit / Delete News Articles
- 📂 Category-Based News Organization
- 🔍 Search Functionality
- 🎥 YouTube Video Integration
- 📰 Breaking News Section
- 🗄️ MongoDB Database Storage
- 📱 Responsive UI Design
